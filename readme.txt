Ultra-fast, Compare.com-style multi-step insurance quote forms for WordPress.

== Description ==

Insurance Quote Wizard is a custom WordPress plugin built for insurance agencies. It creates beautiful, ultra-fast, multi-step wizard forms that collect quote leads with the look and feel of Compare.com.

= Features =

* Ultra-fast frontend (~38KB JS, zero jQuery dependency)
* Compare.com inspired one-question-per-step wizard UI
* Visual drag-and-drop form builder
* 3 pre-built templates: Auto Insurance, Home Insurance, Generic
* Unlimited forms, fields, steps, and conditional logic rules
* Complete entries management with pagination, notes, and CSV export
* EverQuote/QuoteWizard style email notifications
* Customer confirmation emails
* Email template customization with 20+ merge tags
* Email delivery log with resend capability
* Spam protection: honeypot, time-check, rate limiting, reCAPTCHA v3
* WordPress dashboard widget
* REST API with 11 endpoints
* Shortcode and Gutenberg block support
* Mobile-first responsive design
* Print-friendly entry views
* Accessibility: ARIA roles, keyboard navigation, focus management

= Shortcodes =

[insurance_quote_wizard id="1"]
[insurance_quote_wizard slug="auto-insurance"]

== Installation ==

1. Upload the `insurance-quote-wizard` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Quote Wizard in the admin sidebar
4. Use shortcodes to embed forms on any page
